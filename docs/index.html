Project Insights

This project utilizes PySpark to process and analyze large-scale fuel price data across France. By cleaning millions of records, we established a pipeline to visualize spatial and temporal trends. The geospatial analysis (Figure 1 & 2) highlights distinct regional disparities, with prices often fluctuating significantly between highway service stations and rural depots.

Key Findings: The time-series analysis revealed a strong weekly seasonality, where prices tend to adjust in predictable cycles. Using this data, we engineered features such as price lags (t-1, t-7) and rolling averages to train Machine Learning models (Random Forest & Gradient-Boosted Trees) that forecast future prices with high accuracy.
