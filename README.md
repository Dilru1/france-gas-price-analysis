# French Gas Price Analysis with Apache Spark

## Overview
Analysis of gas prices in France (2022-2024) using PySpark. This project creates a data pipeline to ingest, clean, visualize, and forecast gas prices.

## Project Structure
* `src/`: Source code for data loading and transformation.
* `notebooks/`: Jupyter notebooks for visualization and analysis.
* `config/`: Configuration parameters.
